## WX-SPORT

> 背景：牛油果微信公众号。
> 项目地址：  https://avocadomethod.cn/dist/index.html
> github地址：https://github.com/liuyahuiZ/wx-sport


#### 前端技术架构

基于React开发，UI组件是自主研发的neo-ui

flex布局 + scss样式

webpack4 + ReactRoute页面拆分，动态加载
